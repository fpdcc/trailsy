About This Map

How do I get to Busse Woods? Where is the closest trail I can take my bike? Is there a bathroom near my picnic grove?

This map will eventually provide answers to these questions and more, allowing visitors to discover, and travel to, amenities within the Forest Preserves of Cook County.

Right now this map is a prototype, meaning there is still work to be done before it's ready for public consumption. Contact fpdcc.info@cookcountyil.gov for more information about this project.

This project is built using the OpenTrails specification and To The Trails web application. It is made possible with funding from the Centers for Disease Control and Prevention through the Healthy HotSpot initiative led by the Cook County Department of Public Health.
